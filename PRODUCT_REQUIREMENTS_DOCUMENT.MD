This is a structural/tooling bootstrap pass — see
`development-plans/PLAN-chore-bootstrap-uniform-structure.md`. There is no
shipped product yet: the prior experimental `app.py` (a PDF text extractor)
was dropped without being ported forward.

implements it. When that work is planned, this file should be updated (or
replaced) to reflect the reconciled requirements, per the usual
`/requirements-review` process — `DESIGN.md` itself stays as the original
design reference.
